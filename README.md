# CutreGameC64


Welcome to my first video game for Commodore 64. It is not the most optimized game but , it is my first game and I'm proud of the result , I thought the result was worse. It is a simple game, 2 war tanks fighting with them self. 

![fibob](https://github.com/tsw1985/CutreGameC64/blob/main/main.png)

# Compile

1 - Download the kickAssembler compiler : https://theweb.dk/KickAssembler/Main.html#frontpage

2 - Download VICE Emulator: https://vice-emu.sourceforge.io/index.html#download
3 - Install plugin "KickAss" in your Visual Studio Code and set up.

4 - Press Control + Shift + P and select "KickAss (C64) Build and Run

5 - In the VICE emulator , setup the Joysticks with the keys in your keyboard, joystick 1 and 2.

![fibob](https://github.com/tsw1985/CutreGameC64/blob/main/joystick_config_1_2.png)

![fibob](https://github.com/tsw1985/CutreGameC64/blob/main/joystick_config_2_2.png)

6 - Play and enjoy !!
